### Лабораторная работа №3
# Описание
  Программа для тренировки памяти: запоминание и воспроизведение чисел разной длины.

# Требования
  - Python 3.6 или выше
  - Стандартные библиотеки Python (ничего дополнительно устанавливать не нужно)
## Установка и запуск
# 1. Клонирование репозитория
    git clone https://github.com/fedya-greke/ProgaLab3.git   
    cd <папка проекта>
# 2. Создание виртуального окружения (рекомендуется)
  # Для Windows
    python -m venv venv
    venv\Scripts\activate

  # Для Linux/Mac
    python3 -m venv venv
    source venv/bin/activate
# 3. Проверка зависимостей
  Проект использует только стандартные библиотеки Python:
   - tkinter, ttk (основные библиотеки для работы оконного приложения)
   - random (для генерации чисел)
   - traceback (для отладки программы)
   Никакие дополнительные библиотеки (pip install) не требуются!
